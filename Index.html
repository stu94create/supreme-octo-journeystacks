📱 About your data

Your library is stored on this device in Safari. It stays safe as long as you don't clear Safari's website data in iPhone Settings.

For the best experience, save this app to your home screen — tap Share in Safari then Add to Home Screen. On the next screen make sure it says Add to Home Screen and not just Add Bookmark. This tells iOS to treat Stacks like a real app, opens it full screen without the Safari address bar, and makes data loss much less likely.

Back up regularly using the Backup button above. Store the file in iCloud Drive or email it to yourself so you can restore if anything goes wrong.
